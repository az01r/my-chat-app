# Chat app built with nodejs and Flutter

## Useful commands

`npm tsc`

`npm run start`

`npm run dev`

`flutter run`

`flutter run -d chrome --web-port 3000` To run the fronend in chrome you also have to manually set the port for cors checks 

## .env frontend settings

If running on local machine remember to set the correct BACKEND_URL

Web `BACKEND_URL=http://127.0.0.1`

Mobile `BACKEND_URL=http://10.0.2.2`
